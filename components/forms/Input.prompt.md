A labelled single-line text field for forms (booking, intake, contact). Calm sage focus ring; supports `hint` and `error`.

```jsx
<Input label="Full name" placeholder="Your name" required />
<Input label="Email" type="email" error="Please enter a valid email" />
```

Props: `label`, `type`, `placeholder`, `hint`, `error`, `required`, `disabled`. Pair with `Button` and `Select` inside a form column.
